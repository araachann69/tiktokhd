How It Works

FOUR
METHODS,
ONE PRIVATE
FLOW.

Inspects the MP4 container, estimates FPS from sample timing, and writes the selected recipe in the browser. No new frames. No server upload.

timing + elst

Balanced Sync

Changes the main timing headers, then adds a speed guard. Use first for a stronger patch without the old desktop slowdown.

elst + tags

Extension Signal

Adds a normal-speed edit list and iTunes metadata. Does not change playback speed, so the file previews normally.

mvhd

Header Lite

Changes only the outer movie timing. Easier on desktop players, but the upload signal is lighter.

mdhd

Classic Force

Changes movie + video track timing like the original patchers. Useful as fallback, but desktop playback can look slow.
